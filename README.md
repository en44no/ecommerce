## 💻 Run Locally

>Clone the project
>
>```bash
>  git clone https://github.com/en44no/ecommerce.git
>```
>
>Go to the project directory
>
>```bash
>  cd my-project
>```
>
>Install dependencies
>
>```bash
>  npm install
>```
>Create .env file and add your API key from commerce.js
>
>```bash
>  REACT_APP_CHEC_PUBLIC_KEY=yourAPIKey
>```
>
>Start the server
>
>```bash
>  npm start
>```
